# 🚗 Automobile Sales Analysis Dashboard (Power BI)

An interactive **Automobile Sales Analysis Dashboard** built in **Microsoft Power BI** to analyze sales performance and transform raw automobile sales data into actionable business insights. The dashboard leverages **Power Query**, **DAX**, and interactive visualizations to monitor revenue, profit, sales quantity, customer ratings, and regional performance.

---

## 📌 Project Overview

The **Automobile Sales Analysis Dashboard** provides a comprehensive overview of key business metrics through interactive reports and visualizations. It enables users to analyze revenue trends, evaluate regional performance, compare product categories, and monitor profitability, helping stakeholders make informed, data-driven decisions.

---

- 📊 Interactive KPI Cards
